# Scraping Sports Data With R

A short primer on using R to scrape sports data, from easy to hard.

## Requirements

To do this, you'll need the `tidyverse` and `rvest` libraries, which you can get from CRAN.

```
install.packages(c("tidyverse", "rvest"))
```

## Why Scrape Sports Data

It's everywhere and people are interested. Simple. 

## What is in this repository

Three R notebooks with step by step instructions on how to scrape data when it's easy, when it's a little harder, and when it's the NCAA and is awful.
